KumaStream Static Website
=========================

Static website version of KumaStream Blogger template

PROJECT STRUCTURE:
- index.html          Main HTML file
- css/style.css       All styling
- js/main.js          JavaScript functionality

FEATURES:
- Responsive Design
- Dark Mode Toggle
- Anime Slider (Owl Carousel)
- Grid Layout
- Popular Posts Widget
- Genre Filter
- Search Functionality
- Mobile Menu

SETUP:
1. Open index.html in your browser
2. No build process required

DEPLOYMENT:
- Netlify: Drag and drop folder
- Vercel: Run 'vercel --prod'
- GitHub Pages: Push to repo and enable Pages

CUSTOMIZATION:
Edit css/style.css to change colors
Edit js/main.js to modify sample data

Built with HTML, CSS, and JavaScript
